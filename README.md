## Project Overview
This project demonstrates SQL database management and business data analysis using a simulated hotel management database.

The database includes tables for:
- Guests
- Rooms
- Bookings
- Payments

The goal was to answer business and operational questions using SQL queries.

## Key SQL Concepts Demonstrated
- Data insertion and deletion
- Table updates
- JOIN operations
- Aggregation functions
- Conditional ordering
- Business KPI calculations

## Example Business Questions Solved
1. Retrieve confirmed bookings with guest names and room types
2. Calculate average room prices by room category
3. Determine total revenue from confirmed payments
4. Identify rooms requiring maintenance
5. Prioritize rooms by operational status
